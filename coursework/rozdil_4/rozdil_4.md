# РОЗДІЛ 4. ПІДСИСТЕМА МІГРАЦІЙ І CLI

У цьому розділі розглянуто, як YAOI керує еволюцією схеми бази даних у часі: discovery файлів-міграцій, контроль цілісності через SHA-256 чексуми, транзакційне виконання DDL, captured advisory lock для безпеки одночасних запусків та CLI-команди, що пов'язують усе це разом для зручного використання.
