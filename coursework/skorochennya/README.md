# Скорочення та умовні познаки

**Орієнтовний обсяг:** 0.5-1 сторінка.

## Що має містити (за методичкою)

Перелік усіх скорочень, абревіатур та умовних позначок, що використовуються у тексті роботи, з їх повним розшифруванням.

## Конвенція ведення

Файл [skorochennya.md](./skorochennya.md) ведеться інкрементально: при написанні кожного нового розділу всі введені там скорочення одразу додаються сюди в алфавітному порядку. Це гарантує, що до моменту складання фінального документа перелік буде повний і узгоджений із текстом.

## Порядок розташування у роботі

За методичкою — між **Змістом** та **Вступом** (структурна частина №4 з 9).
