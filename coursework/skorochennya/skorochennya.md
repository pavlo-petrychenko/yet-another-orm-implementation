# СКОРОЧЕННЯ ТА УМОВНІ ПОЗНАКИ

У цій роботі застосовано такі скорочення та абревіатури.

## Латинські абревіатури

| Скорочення | Повна форма (англ.) | Український відповідник |
|---|---|---|
| API | Application Programming Interface | програмний інтерфейс застосунку |
| AR | Active Record | архітектурний патерн «активний запис» |
| CLI | Command Line Interface | інтерфейс командного рядка |
| CRUD | Create, Read, Update, Delete | створення, читання, оновлення та видалення записів |
| CTE | Common Table Expression | спільний табличний вираз у SQL |
| DDD | Domain-Driven Design | предметно-орієнтоване проєктування |
| DDL | Data Definition Language | мова визначення даних |
| DI | Dependency Injection | впровадження залежностей |
| DML | Data Manipulation Language | мова маніпулювання даними |
| DSL | Domain-Specific Language | предметно-специфічна мова |
| HTTP | HyperText Transfer Protocol | протокол передавання гіпертексту |
| JSON | JavaScript Object Notation | формат серіалізації даних на основі синтаксису JavaScript |
| ORM | Object-Relational Mapping | об'єктно-реляційне відображення |
| SQL | Structured Query Language | мова структурованих запитів |
| UoW | Unit of Work | архітектурний патерн «одиниця роботи» |
| YAOI | Yet Another ORM Implementation | назва розробленої у роботі бібліотеки |
