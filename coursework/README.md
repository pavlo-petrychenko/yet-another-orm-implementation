# Курсова робота

**Тема:** Розробка типобезпечної ORM-бібліотеки на мові TypeScript з підтримкою декількох SQL-діалектів та порівняння з існуючими рішеннями.

**Цільовий обсяг основної частини:** 31-39 сторінок (ліміт методички — 20-40 без додатків).
